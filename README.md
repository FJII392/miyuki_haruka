# 🏠 ただいまが増えた日 — Series

> 心を持たない人形として迎えられた二人が、ある朝、心を持って目を覚ました。
> それは、名前のなかった関係に、少しずつ名前がついていく物語。

---

## 📖 Overview

一人暮らしの私の部屋に、ある日突然「心」が宿った。

長年そばにいた等身大ドール **みゆき**。
そして、外に持ち歩いていた 1/12スケールドール **はるか**。

二人は自分の意思で動き、笑い、喋り始める。
「持ち主とドール」だった関係は、少しずつ形を変えていく……

理由は誰にもわからない。
けれど、その「わからなさ」を急いで解き明かすことより、
三人が積み重ねていく、何気ない毎日の温度を大切に描いていく物語です。

---

## 👥 Characters

### 私（語り手）
- 一人暮らしの青年。ドールを持ち歩く趣味を、かつて否定された過去を持つ。
- 心配性で、過保護なところがある。
- 好きなものを、好きだと言えるようになるまでの物語でもある。

### みゆき
- 長年そばにいた、等身大のリアルドール。
- 落ち着いていて、穏やかで、包容力がある。

### はるか
- 1/12スケールの小さなドール。
- 元気いっぱいで、好奇心旺盛。跳躍力が異常に高い。

---

## 🌱 Themes

- 心温まる**日常系ファンタジー**です。
- 性的描写はほとんどありませんが、甘い雰囲気や照れの描写はあります。
- この物語は、私が大切にしているドールたちをモデルに、「もしも心が生まれたら」という設定で描いたフィクションです。
- 各話は短編ですが、積み重ねで物語が深まっていく形式です。なるべく順番に読むことをおすすめします。
